Mission Statement

If you try hard and fail to understand some concept, it is not because concept is too difficult for you, it is just that the idea is poorly presented.

Despite great amount of documentation it is more and more difficult task to track ever increasing amount of techology information. Some projects are better documented, some are poorly documented, part of documentation is outdated, part of it is just too much to read.

My mission is to present in simple, quick and easy to understand words a wide range of topics from Django Web Framework ecosystem. Subjects like deployments, python programming, message queues are conffered in human language with lots of pictures, realistic usecases and to the point videos.

I prepare meticulously each lesson to save your time on learning, help you understand swiftly and apply learned skills in your projects.

Eugen Ciur, Django Lessons

For Mid-Level Developers

After you skimmed Django Documentation, watched a couple of YouTube videos, completed a few simple projects, landed in couple of python jobs - this is where it starts to be challenging to improve your skills. You will realize (like me) that there are less and less resources for intermediate developers. Django lessons target this niche.

Not for Noobs

I always felt that that beginners are lucky. Out there are a great amount of tutorials, videos, books which explain how to implement a quicksort algorithms, how to define a python's class or what is a model. Beginners have plenty of material to learn basics from.

So if you are just staring to learn IT and programming in general, probably Django Lessons is not for you. As a general rule, I will not explain what is a function in python, what is a HTTP request or that well known port for ssh is 22, for http is 80 and https is 443 - I take those things as obvious
